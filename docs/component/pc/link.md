## 属性
|  属性   | 说明  | 类型  | 可选值  | 默认值  |
|  ----  | ----  | ----  | ----  | ----  |
| type  | 类型 | string | primary / success / warning / danger / info / text | — |
| underline  | 是否下划线 | boolean | — | true |
| disabled  | 是否为禁用状态 | boolean | — | false |
| href  | 原生 href 属性 | string | — | — |
